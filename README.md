# Beacon Daily

Beacon Daily is a CRUD app that allows users to:

- Create new checklist items
- View both complete and incomplete tasks
- Update tasks with new information
- Delete both complete and incomplete checklist items

### Technologies Used

- React (Hooks)
- React Router
- PostgreSQL
- Node.js
- Express
- Material UI
